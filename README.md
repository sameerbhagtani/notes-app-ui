# Notes app UI

This project is a simple Notes app UI built with React Native and Expo. It focuses on a clean, modern interface for browsing and managing notes across mobile screen sizes. The app is structured with reusable components and modular feature folders to keep the codebase easy to maintain.

This project is part of my [ChaiCode Mobile Dev Cohort 1 Archive](https://github.com/sameerbhagtani/mobile-dev-cohort-1). Checkout my entire journey there!

---

## Features

- Dark and light mode toggle with support for system theme preferences.
- Responsive layout behavior for phones and larger device sizes.
- Create, edit, and delete notes from a simple and intuitive interface.

## Local Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/sameerbhagtani/notes-app-ui.git
cd notes-app-ui
```

2. Install dependencies:

```bash
npm install
```

3. Start the Expo development server:

```bash
npm start
```
